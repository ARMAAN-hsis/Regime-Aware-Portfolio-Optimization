# Regime-Aware Portfolio Optimization Engine
**Category:** Financial Engineering / Machine Learning (PBL-2)

## Overview
This project develops an intelligent investment system that uses Machine Learning (Hidden Markov Models) to identify market regimes (Bull, Bear, Volatile) and dynamically adjust portfolio allocation strategies.

## Key Features
- **Regime Detection:** ML-based classification of market states.
- **Adaptive Optimization:** Switching between Max Sharpe and Risk Parity models.
- **Risk Management:** Focus on minimizing Maximum Drawdown.

## Tech Stack
- Python (Pandas, NumPy, Scikit-Learn)
- Financial Data (YFinance, FRED API)
- Optimization (CVXPY)
